Scientific Calculator in C

This project is a Scientific Calculator built in C, capable of performing both basic arithmetic and advanced scientific operations. It supports both integer and floating-point calculations.

The calculator includes the following features:

  • Basic Arithmetic: Addition, Subtraction, Multiplication, Division
  • Advanced Scientific Operations: Trigonometry (sin, cos, tan), Logarithms (log, ln), Exponentiation (x^y)
  • Support for Integer and Floating-Point Calculations
  • Console Interface: Simple, user-friendly command-line interface

The calculator provides an easy-to-use console interface for the user to input mathematical expressions, making it ideal for both basic and scientific calculations.

Key Features:

  1. Arithmetic Operations: Addition, subtraction, multiplication, and division.
  2. Trigonometric Functions: sin, cos, tan, and their inverses (asin, acos, atan).
  3. Logarithmic Functions: log (base 10), ln (natural logarithm).
  4. Exponentiation: Ability to raise a number to any power.
  5. Support for both Integer and Floating-Point Numbers
  6. Simple Command-Line Interface

To run the calculator, compile the C program and run it via your console. Input expressions as requested, and the program will output the result accordingly.

Example Usage:

$ gcc calculator.c -o calculator
$ ./calculator

Enter operation (+, -, *, /, sin, cos, tan, log, ln, exp): sin
Enter value (in radians): 1.5708
Result: 1

This project provides a foundation for implementing more advanced features, and can be expanded further with additional scientific functions, error handling, or a more complex user interface in the future.
